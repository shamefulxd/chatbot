Функциональные требования к нашему проекту
1. Получение рецептов по ингредиентам:

   • Пользователь вводит список имеющихся ингредиентов.

   • Бот анализирует список и предлагает рецепты, которые можно приготовить с указанными ингредиентами.

2. Получение рецептов по категориям:

   • Пользователь выбирает категорию (вегетарианские, десерты, супы и т.д.).

   • Бот предоставляет список рецептов в выбранной категории.

3. Поиск рецептов:

   • Возможность поиска рецептов по ключевым словам (например, "шоколад", "курица").

4. Сохранение избранных рецептов:

   • Пользователь может сохранять понравившиеся рецепты в избранное для быстрого доступа.

Технические детали

1. Язык программирования: Python

2. База данных: Использование базы данных для хранения рецептов, ингредиентов и пользовательских данных.

3. API: Разработка API для взаимодействия между ботом и базой данных.
